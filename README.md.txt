# Food Freshness Classifier (AI Capstone Project)

This project uses a Convolutional Neural Network (CNN) trained in Python (TensorFlow + Streamlit) to classify food images as:

- **Fresh**
- **Stale**
- **Spoiled**

### 🔍 Purpose
To help reduce food waste and promote SDG 12 – Responsible Consumption and Production.
